## Сборка из исходников

#### Шаг 1. Установка gradle

Скачать последнюю версию cradle отсюда `https://gradle.org/releases`

#### Шаг 2. Распаковка

```shell
$ mkdir /opt/gradle
$ unzip -d /opt/gradle gradle-8.1-bin.zip
$ ls /opt/gradle/gradle-8.1
LICENSE  NOTICE  bin  getting-started.html  init.d  lib  media
```

#### Шаг 3

```shell
$ export PATH=$PATH:/opt/gradle/gradle-8.1/bin
```

#### Шаг 4. Сборка APK

Из директории проекта запустить команду

```shell
$ ./gradle
```

Путь APK файла: `.../BestBurgers/app/build/outputs/apk/debug/app-debug.apk`

## Описание и функционал

#### Эран приветствия

- Логотип.
- Новости на тему здоровья.
- Кнопка входа "Сделать заказ".

#### Эран "Заказ бургеров"

- Сверху навигационная панель.
- Список бургеров для заказа.
- Кнопки "В карзину". При нажатии товар попадает в Корзину.

#### Эран "Заказ напитков"

- Сверху навигационная панель.
- Список напитков для заказа.
- Кнопка "+" добавляет в Корзину тот напиток, напротив которого она установлена.

#### Эран "Корзина"

- Сверху навигационная панель.
- Список добавленных товаров.
- Общая стоимость товаров.
- Кнопка "Очистить" обнуляет список товаров.
  - Перед обнулением у пользователя запрашивается, уверен ли он в своих действиях.
    - Если Да, то список обнуляется.
    - Если Нет, то список не обнуляется.
- Кнопка "Оплатить" обнуляет список и сообщает пользователю о сделанном заказе и с ним свяжется оператор.

#### Эран "Контакты"

- Сверху навигационная панель
- Список контактов
- При открытии экрана, карта плавно приближается к адресу ресторана, где уазан график работы.
